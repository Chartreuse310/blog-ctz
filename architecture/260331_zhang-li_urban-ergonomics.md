# 城市人因：阅读与设计高质量生活

建筑师是做什么的？在房地产建设过剩的今天，建筑师应该做什么？

——“衣食住行玩”的需求，从未停止过。人类从未停止营造家园，从生存到==栖居==，人类通过和家园间的关系，建立一种生活记忆的印记。这种需求是建筑学存在的原因。

## 1. 存量提质的时代背景

- 时代背景：高质量城镇发展（“十五五”）
- 设计转型：从“形容词”到“数据”
- 可量化与不可量化：从聚焦于“物”到聚焦于“人”，空间体验质量从“底线”到“优线”，==根据具体情形提供体验==而非均匀达到指标


|     | 形容词 | 数据     |
| --- | --- | ------ |
| 文化  | 身份  | 文化批评生成 |
| ↑   | 愉悦  | 审美判断生成 |
| ↓   | 舒适  | 感知生成   |
| 功用  | 便利  | 行为生成   |


## 2. 城市人因：基于空间体验量化而设计

- 城市人因研究：从问卷到数据，范式的转化
  - 数据从何而来？从面部表情到身体姿态
  - 基本任务：==识别、漫游、共享、体感== [^1]
    - 识别：重要建筑物（目前仅视觉，气味儿无法模拟——但人与人的感受受气味儿影响很大）
    - 漫游：人类往往愿意走喜欢的路
    - 共享：视线交互反映交流的发生与强度（学生喜欢和真人交流的教学，但AI教出来的学生得分更高）
    - 体感：身体与物质的互动，拟人化导致人类总会将看到的东西映射到人的身体（如将猫的脸当成人、将建筑物的顶当成脸所以更重视屋顶）

## 3. 来自设计实践 增进设计实践

- 人因测度数据：量化空间体验维度
  - 建筑是人类生活最大的数据集
  - 多人因测度的采集：判断不同建筑方案的空间体验
- 人因分析技术路径：在传统设计流程中==增加“中试平台”==
  - **首钢大跳台**：形象&艺术创意来源于人的想法（传统设计），朝向（影响天际线）由量化分析得到，建成后分析运动员反馈、设计冷却塔标识位置
  - **雪如意**：形象&艺术创意来源于人的想法（传统设计），开口位置使用VR测试（先分析对VR的适应程度，再测试相对的停留时长）
  - **三山五园艺术中心**：开口问题使用分析法，发现人们更喜欢远处的圆弧（与雪如意愿景视觉停留相似）。
  - **威尼斯双年展主展区**：机器人捕获行为辅助策展（预测观众行为，对展览效果打分），机器人也增加了交互。
  - **上海人民广场地铁站**：出口编号非地理位置而是建设顺序，发现人在找路时习惯于看前上15°的方向，人在地铁站倾向于不停下来所以需要提前提示路径方向
  - **教室更新装饰界面**：采集空间声、光环境数据，识别空间问题，制定设计框架。用Stroop测试墙面颜色对集中力的影响，发现树林照片类似白噪声效果。
  - **清华大学学生活动中心**：全地下开发，整体设计决策来源于人（传统设计），发现微微晃动的树影让人感觉处于室外。
  - 城市人因设计决策推演平台——**清华大学一站式服务大厅**：测试工位数量和位置的影响
  - **上海街区**：测试不同决策的影响

建筑学院系馆有很多==反例==，如声学空间问题。
