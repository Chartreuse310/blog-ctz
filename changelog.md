# 更新日志

## 1.1.0 (2026-05-21)

🔍 查看全部页面

- 新增"查看全部"页面，支持按年份和标签筛选所有文章

- 移除首页 Tags 统计卡片
- 新增最近 30 天更新频率折线图（平滑曲线 + 渐变填充）
